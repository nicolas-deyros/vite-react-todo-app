# React Todo APP using Vite

This is a simple todo app built with React.js and Vite.js.

Tecnologies used:

- React.js
- Vite.js
- Framer Motion
- React Icons
- React Toastify
- UUID

## Installation

```bash
  npm i

  npm run dev
```
